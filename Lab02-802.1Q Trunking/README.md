Lab 2: Router-on-a-Stick Inter-VLAN Routing (802.1Q Trunking)

Concept Demonstrated
Router-on-a-Stick (802.1Q Trunking) is a method of enabling inter-VLAN communication using a single physical router interface divided into logical subinterfaces. Each subinterface handles traffic for a specific VLAN using 802.1Q encapsulation. This lab demonstrates how switches use VLANs to segment networks and how a router provides Layer 3 routing between them through a trunk link.
